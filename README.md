# 🌾 AgroData IA - Gestão Agrícola

Aplicativo Streamlit para gestão financeira de fazendas com análise por IA.

## ✨ Funcionalidades
- 📊 Dashboard financeiro com métricas em tempo real
- ✏️ Editor de dados interativo
- 📥 Exportação para CSV
- 📈 Gráficos automáticos
- 🤖 Análise com IA Gemini

## 🚀 Como executar localmente

1. Clone o repositório
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
Configure a chave da API Gemini:

bash
export GEMINI_API_KEY="sua_chave_aqui"
Execute o app:

bash
streamlit run projeto2_agro.py
☁️ Deploy no Streamlit Cloud
Conecte seu repositório GitHub

Configure a chave API nos Secrets:

toml
GEMINI_API_KEY = "sua_chave_api"
Clique em Deploy!
